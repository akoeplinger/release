Mono 1.0.5 and 1.1.3 Released

Updates to the production and development versions of Mono have been released.

Follow the links above for the details. The major features of this release are:

  • Generics support in the compiler is now feature complete. Use the bundled Mono.C5 a library to explore the generics world.
  • Couple more features from C# 2.0 are implemented.
  • SSAPRE optimization engine: for your performance needs.
  • New simplified build system, single `make' compilation of C and C# sources.
  • New and improved Mono.Unix namespace.
  • Faster, leaner, stabler everything.
  • .exe files are now gone from your /bin directory.
