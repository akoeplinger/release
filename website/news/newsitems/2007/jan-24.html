Mainsoft Releases Grasshopper 2.0 Preview

Mainsoft has released a preview of their 2.0 Grasshopper product.

Grasshopper allows server-side .NET applications to run in a J2EE server. This is done by recompiling the CIL bytecodes into JVM bytecodes and providing a Mono-based set of class libraries for the applications to run.

Major new features in this new release include ASP.NET 2.0 support and Visual Studio based debugging.
