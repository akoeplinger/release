Resco MobileTouch Toolkit

Resco has just announced the release of their Resco MobileTouch Toolkit, a set of UI controls for MonoTouch based on their Resco MobileForms Toolkit’s controls, enabling application development for iPhone and iPad in a similar model to standard .NET Compact Framework development.

If you have an iPhone or iPad, you can see their MonoTouch-based controls in action on device by downloading the Resco MobileCRM Lite application.

More details can be found on the Resco MobileTouch Toolkit website.
