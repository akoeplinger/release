Run Locally in Mono
Step 1:

Open the BlogEngine solution in Visual Studio
Step 2:

In the Mono menu, click "Run Locally in Mono".
Step 3:

If you don't have Mono for Windows installed, a dialog message like this will appear. Install Mono for Windows, then proceed.
Step 4:

Windows firewall will display a dialog saying that mono.exe has been blocked from opening a port in the firewall. Click "Unblock".
Step 5:

BlogEngine will be launched in your default browser.
Step 6:

To stop this instance of BlogEngine, right click on the MonoVS tray icon and click "Stop".
