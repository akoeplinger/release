This README is for daily build and daily Tests 

Daily Build 
===========
scripts used 
1.  mono-snapshot.sh (daily build and tar-balls)
2.  libgdiplus-snapshot.sh 
3.  make-testresults.pl (run test)
4.  store-testresults.pl (store test results in one xml)
5.  generate-consolidated-charts.pl (generate charts)
6.  generate-html.php (static html pages)
7.  putfilesonserver.sh (ssh all png, html to go-mono)
8.  displayTestResults.php (display results )
9.  displayDetails.php (details of test results)
