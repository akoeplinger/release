Get the bb source tarball (TODO: add link to tarball of 1.6 branch and my
additional patches, files)

Add distro and/or arch to conf/os.conf if it's not there

Copy all files (if working from a subversion work area):
svn export . bb

tar up bb and copy to build machine

Extract, then:

perl Makefile.PL
make
sudo su
make install

Install XML::Simple (or XML::Parser if using the bb 1.6 branch)
Possibly from cpan...

Inside jail, from inside the unpacked tarball, as root (copy and paste):

---------------------
export DEST=/usr/share/ximian-build-system

mkdir -p $DEST/conf

cp -f conf/os.conf $DEST/conf
cp -f conf/bb.conf $DEST/conf
cp -f conf/packsys.conf $DEST/conf

cp -Rf packsys $DEST

mkdir -p $DEST/scripts
cp -f scripts/distribution.guess $DEST/scripts
cp -f scripts/config.guess $DEST/scripts
---------------------
