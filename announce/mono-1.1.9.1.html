Mono 1.1.9.1 Release Notes

Mono 1.1.9.1 is a minor bug fix release to Mono 1.1.9, for details on 1.1.9 see its release notes.

Important Bug Fixes in 1.1.9.1

Various bugs reported on the new ASP.NET stack have been fixed (DataGrid, TableColumn, BoundColumn, javascript, HttpResponse.Send).

ASP.NET no longer crashes if GDI+ is missing.

msxsl:script regression fixed.

Generics Dictionary wont crash when initialized with zero capacity.

mod_mono and xsp: cope better with errors under Apache 1.3

Eliminates defunct processes after using the Process class.

Bug fix for async sockets.

Fix SoapReflectionImporter in the serializer.

Distributes the Firebird database provider.

Availability

Source code and packages are available from: http://www.mono-project.com/Downloads
