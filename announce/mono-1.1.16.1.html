Mono 1.1.16.1 Release

This is a minor update to the just released Mono 1.1.16 and it contains some minor fixes.

  • What is new since the last release.
  • Download and installation.

This release is mostly a bug-fix release, there are very few new developments.

Changes since Mono 1.1.16

Fix Windows.Forms' DataGrid regressions (Chris).

Improve focus handling in Windows.Forms (Jackson, Peter).

Fix RTF parser regression in Windows.Forms (Peter).

Fix LinkLabel (Chris).

Bug fix in Double.Parse (Zoltan).

Bug fix in DynamicMethod, to support IronPython Beta 8 (Zoltan).

Fix handling of SpecialNameAttribute (Zoltan).

Properly dispose a number of graphics objects in System.Drawing (Image, Pen, SolidBrush) (Sebastien).

FileDialog now loads directories asynchronously (Alexander).

Installing Mono 1.1.16.1

	
	$ ./configure --prefix=/devel

You can then setup your PATH to include /devel/bin to access the Mono 1.1. Alternatively you can replace your Mono installation with 1.1.16.1

Binary Packages and Source Code Downloads:

Source code and pre-compiled packages for SUSE, SLES, Fedora Core 3, 4, RHEL, MacOS and Windows in a variety of platforms available from our web site from the download section.

Quick source code installation:

If we have no packages for your platform, installing from source code is very simple.

mono:

	
    $ tar xzf mono-1.1.16.1.tar.gz
    $ cd mono-1.1.16.1
    $ ./configure
    $ make
    $ make install

Then compile libgdiplus: 

	
    $ tar xzf libgdiplus-1.1.16.1.tar.gz
    $ cd libgdiplus-1.1.16.1
    $ ./configure
    $ make
    $ make install
