Welcome to the Mono Installer @@BITROCK_MONO_VERSION@@


The Mono Installer provides you with the easiest way to install and run the Mono platform in a variety of Linux distributions.

Mono is a comprehensive open source development platform based on the .NET framework that allows developers to build Linux and cross-platform applications with unprecedented productivity. Mono's .NET implementation is based on the ECMA standards for C# and the Common Language Infrastructure.

For further information, please visit the Mono Project website.



Supported Platforms


The goal of the Mono Installer is to run in the same platforms that Mono does. This release has been tested on the following x86-based Linux distributions:

  • Red Hat 9.0
  • Fedora Core 1, 2, 3
  • Suse Enterprise Server 9
  • Suse Linux 9.0
  • Novell Linux Desktop 9

The binaries have been built on Red Hat 9, so they will not work with older platforms with incompatible GLIBC, such as Suse Enterprise Server 8.0. If you are using Mandrake 10, you may need to set the following environment variable : 

MOZILLA_FIVE_HOME=/usr/bin

Support for Windows and OSX is planned.



Getting Started

The same Mono binary installer will work across all Linux Distributions. The first step is to download the installer file and add executable permissions:

chmod +x mono-@@BITROCK_MONO_VERSION@@-linux-installer.bin

You can now execute it either by double clicking it from your desktop environment or launch it from the command line:

./mono-@@BITROCK_MONO_VERSION@@-linux-installer.bin

You could also launch the installer in text or unattended modes:

./mono-@@BITROCK_MONO_VERSION@@-linux-installer.bin --mode text

./mono-@@BITROCK_MONO_VERSION@@-linux-installer.bin --mode unattended

After this, follow the on-screen instructions. By default, the software will be installed in the ~/mono-@@BITROCK_MONO_VERSION@@ directory. An icon will be created in your desktop environment that will allow you to invoke the different applications such as Mono Doc, Mono Develop as well as to uninstall.



Components

The Mono Installer includes the following components.

Mono Runtime: Mono includes a compiler for the C# language, an ECMA-compatible runtime engine (the Common Language Runtime, or CLR),and class libraries.

ADO.NET: Including support for MySQL, SQLite, IBM DB2, Sybase, PostgreSQL and Oracle databases.

ASP.NET: Web development. Includes XSP

GTK#: GTK# is an event-driven system windowing library. This library allows you to build fully native Gnome application using Mono and includes support for user interfaces built with the Glade interface builder.

Mono Develop: An integrated development environment for creating Mono applications.

MonoDoc: A scalable documentation browser and editor developed especially for Mono and .NET

IKVM: A Java VM for .NET

Windows Forms: Preview release of System.Windows.Forms
